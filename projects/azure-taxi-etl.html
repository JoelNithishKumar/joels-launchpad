📊 Azure Yellow Taxi Data Engineering

Medallion Architecture ETL Pipeline on Microsoft Azure

📌 Project Summary

Built an end-to-end Azure-based ETL pipeline using the Medallion architecture (Bronze → Silver → Gold) — increasing data accuracy by 30% and enabling business intelligence dashboards in Power BI.

⚙️ Solution Architecture

  • Azure Data Factory orchestrates ETL from raw Yellow Taxi CSVs
  • Azure Data Lake for staging (Bronze) and curated zones (Silver, Gold)
  • Azure Synapse SQL for modeling and aggregation
  • Power BI dashboards for KPIs (fare revenue, trip count, peak time analysis)

🧩 Tech Stack

  • Azure Data Factory, Data Lake, Synapse Analytics
  • Power BI, DAX
  • PySpark, Pandas for preprocessing
  • SQL, GitHub Actions (CI)

📈 Results & Impact

  • Data accuracy improved by 30%
  • Recognized as Best Project among 50+ submissions
  • Reduced data prep time from days to hours

📌 My Role

Data Engineer — built the ETL, modeled curated tables, designed KPIs in Power BI, and automated quality checks.
